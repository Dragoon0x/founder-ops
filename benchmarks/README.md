# Benchmarks Library

Real-world metric benchmarks by industry and stage. Use these to:
- Calibrate your numbers against reality (not aspiration)
- Set targets for next quarter / next round
- Spot when something is off (yours vs benchmark gap)

**These are reference points, not laws.** Your business may legitimately
diverge — but if you're 2x off a benchmark, ask why.
