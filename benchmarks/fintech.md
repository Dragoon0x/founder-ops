# Fintech Benchmarks

Fintech operates under different physics: regulation, capital efficiency,
fraud rates, and unit economics that depend on customer behavior.

## Neobank Benchmarks

| Metric | Healthy |
|--------|---------|
| CAC per primary deposit account | $25-100 |
| % funded accounts (have $) | 60-80% |
| % primary accounts (paycheck deposited) | 30-50% of funded |
| Avg deposit balance | $500-3,000 (consumer), $5K-50K (SMB) |
| Interchange revenue per active card user | $30-80/year |
| Subscription revenue per premium user | $50-150/year |
| 30-day active rate | 50-70% |
| Annual churn | 25-40% |

**Path to profitability:** Most consumer neobanks need 4-7 years to break-even.
Revenue per user must reach $80-200/year to cover all costs.

## Lending (Consumer)

| Metric | Healthy |
|--------|---------|
| Approval rate | 30-50% of applicants |
| Default rate (prime) | 2-5% |
| Default rate (near-prime) | 5-10% |
| Default rate (subprime) | 10-25% |
| Net charge-off rate | 3-8% |
| Net interest margin | 5-12% |
| CAC per funded loan | $50-300 |
| Repeat loan % within 12mo | 30-60% |

## Lending (SMB)

| Metric | Healthy |
|--------|---------|
| Average loan size | $25K-$250K |
| Default rate | 4-12% |
| APR range | 8-30% (term loans) |
| Origination time | <72 hours (vs 30-60 days banks) |
| CAC per funded loan | $300-2,000 |

## Payments

**Card processing:**
- Take rate: 2.6-3.5% + $0.10-0.30 per transaction
- Net take rate (after interchange): 0.3-0.6%
- Margin per processed dollar: 0.1-0.3%

**Cross-border / FX:**
- Take rate: 0.5-3% on FX spread
- Volume per active business: $5K-$500K/month

## Insurance Tech

| Metric | Healthy |
|--------|---------|
| Loss ratio | 60-75% |
| Combined ratio | <100% (profitable underwriting) |
| Renewal rate | 80-90% |
| CAC per policy | $100-1,000 |
| Avg premium | $100-3,000/year |

## Crypto / Web3

**Centralized exchanges:**
- Take rate: 0.1-0.5% per trade (volume-discounted)
- Active traders: 5-20% of accounts trade in any month
- Avg trade size: $100-10,000

**Wallet products:**
- DAU/MAU: 15-30%
- Revenue per active: $5-50/month (highly variable)

## Regulatory Cost

Fintech adds significant fixed compliance cost:
- Compliance officer: $150K-$300K/year base
- Legal: $200K-$500K/year for emerging fintech
- Audit + SOC 2 + state licensing: $100K-$500K/year
- BaaS/sponsor bank fees: $5K-$50K/month + per-transaction

**Plan for $500K-$1.5M/year in compliance overhead** at any meaningful scale.

## State of Compliance Required

| Stage | Required |
|-------|----------|
| MVP | Privacy policy, ToS, KYC vendor |
| First customers | Money transmitter exemption analysis or sponsor bank |
| Growth | Money transmitter licenses (state-by-state, $1-5K each, surety bonds) |
| Scale | Federal charter consideration, BSA/AML program, fraud team |

State-by-state MTL is a 12-24 month, $1-3M endeavor.

## Fraud Rates

| Vertical | Fraud Rate (% of transactions) |
|----------|-------------------------------|
| Card processing (online) | 0.1-0.5% |
| P2P payments | 0.05-0.2% |
| Crypto on/off ramp | 0.5-2% |
| BNPL | 1-3% |
| New account creation | 5-15% (without strong KYC) |

If you don't measure fraud rate, your unit economics are wrong.
