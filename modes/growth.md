# Mode: Growth

## Purpose

Design growth experiments, evaluate channels, analyze funnel metrics.
Focus on repeatable, measurable experiments — not vanity growth hacks.

## Submodes

### 1. Channel Evaluation

For any proposed channel, score:

| Dimension | 1-10 Score | Notes |
|-----------|----------|-------|
| Addressable audience | | How many potential customers |
| Targeting precision | | Can you reach ICP specifically |
| CAC potential | | Realistic cost per acquisition |
| Time to signal | | Weeks to know if it's working |
| Scalability | | Can it 10x without breaking |
| Competition | | How crowded is this channel |
| Founder fit | | Can YOU execute on this channel |

Total: /70

**Rule**: Try only channels scoring 45+. Below that, you'll burn time.

### 2. Experiment Design

Every growth experiment needs:
```
## Experiment — [Name]

### Hypothesis
"If we [action], then [metric] will [direction] by [amount] because [reason]"

### Success Criteria
- Primary metric: [specific number]
- Timeline: [exact duration]
- Minimum sample size: [calculated for statistical significance]

### What we're giving up
- [Opportunity cost]
- [Team time: N person-days]

### Kill conditions
- If [leading indicator] after [time], stop and pivot
- Max budget: [$]

### Measurement plan
- Before state: [baseline metric with source]
- During: [check-ins at days X, Y, Z]
- After: [final measurement method]
```

### 3. Funnel Analysis

Standard funnel:
```
Awareness → Interest → Evaluation → Trial → Purchase → Retention → Referral
```

For each stage, identify:
- **Current conversion rate**
- **Benchmark** (industry standard)
- **Biggest drop-off stage** (= priority to fix)
- **Hypothesis for improvement**

### 4. CAC / LTV Analysis

```
CAC = Total spend / New customers acquired
LTV = ARPU × Gross margin × (1 / Monthly churn)
LTV:CAC = Should be 3:1+ for healthy growth
Payback period = CAC / Monthly margin per customer (target < 12 months)
```

### 5. Growth Channel Playbooks

**SEO** (12-24 month horizon):
- Target: 1-3 keywords with 500-5K monthly searches
- Approach: Build 1 definitive resource per keyword, not 100 thin posts
- Signal: Organic traffic doubling monthly for 6 months

**Content/Social** (3-12 month horizon):
- Target: Build audience in ONE channel first, expand later
- Approach: Share in-progress building (not polished announcements)
- Signal: Inbound DMs from potential customers

**Cold Outreach** (immediate):
- Target: Specific ICP with clear pain
- Approach: <100 highly personalized emails, not mass sends
- Signal: 10%+ reply rate, 3%+ meeting rate

**Partnerships** (6-18 month horizon):
- Target: Complementary (not competing) companies serving same ICP
- Approach: Start with co-marketing, escalate to integrations
- Signal: One partnership drives 20%+ of new signups

**Paid Ads** (last resort at early stage):
- Only when: Organic is working and you want to scale
- Target: Channels where you already have proof
- Signal: Payback period < 12 months

## Growth Anti-Patterns

| Pattern | Why It Fails | Fix |
|---------|-------------|-----|
| "Hack" mentality | Unrepeatable, burns out | Build systems, not hacks |
| Running 10 experiments at once | No signal from noise | 1-2 experiments at a time |
| Copying a competitor's channel | Their audience ≠ yours | Test on small sample first |
| Not measuring LTV | CAC looks fine, unit economics broken | Track LTV:CAC, not just CAC |
| Scaling before PMF | Expensive, hides real problems | Prove repeatability on small scale |
