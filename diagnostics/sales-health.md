# Diagnostic: Sales Health

Run quarterly. Takes 2 hours with data access.

## Scorecard

Rate each dimension 1-5 (1 = critical, 5 = excellent):

### Pipeline Health
- Pipeline coverage ratio (pipeline / quota): Target 3-4x. Score 5 if > 4x, 1 if < 2x.
- Pipeline source diversity: Score 5 if no single source > 40%, 1 if one source > 70%.
- Pipeline aging: Score 5 if < 10% of pipeline is > 2x average cycle length, 1 if > 30%.
- Stage conversion rates: Score 5 if improving QoQ, 1 if declining in 2+ stages.

### Rep Performance
- Quota attainment: Score 5 if > 80% of reps hit quota, 1 if < 50%.
- Ramp time for new reps: Score 5 if < 3 months to first deal, 1 if > 6 months.
- Rep turnover: Score 5 if < 15% annual, 1 if > 30%.
- Activity quality: Score 5 if reps spend > 60% time selling, 1 if < 40%.

### Deal Quality
- Average deal size trend: Score 5 if increasing QoQ, 1 if declining.
- Discount frequency: Score 5 if < 20% of deals discounted, 1 if > 50%.
- Multi-threading: Score 5 if > 3 stakeholders per deal, 1 if single-threaded.
- Win rate by competitor: Score 5 if > 40% win rate vs top competitor, 1 if < 20%.

### Process & Tools
- CRM data hygiene: Score 5 if > 90% of fields complete, 1 if < 60%.
- Forecast accuracy: Score 5 if within 10% of actual, 1 if > 30% off.
- Sales playbook exists and is used: Score 5 if updated quarterly and referenced daily, 1 if none exists.
- Sales/marketing alignment: Score 5 if lead quality complaints < 1/month, 1 if constant friction.

## Scoring
- 65-80: Excellent. Maintain and optimize.
- 50-64: Good. Address yellow areas proactively.
- 35-49: Needs work. Prioritize the lowest-scoring dimensions.
- < 35: Critical. Major overhaul needed. Consider external help.

## Common Root Causes by Score Range
- < 35: Usually a leadership problem. Wrong VP Sales, no process, or wrong ICP.
- 35-49: Usually a process problem. Playbook gaps, CRM neglect, or misaligned incentives.
- 50-64: Usually an optimization problem. Coaching, enablement, or tool improvements.
