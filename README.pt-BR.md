# Founder-Ops

> [English](README.md) · [Español](README.es.md) · **[🇧🇷 Português]** · [日本語](README.ja.md) · [한국어](README.ko-KR.md)

**Sistema operacional com IA para fundadores.**

40 modos especializados cobrindo todo o trabalho de fundador: descoberta de
clientes, priorização, captação, crescimento, contratação, finanças,
posicionamento, conteúdo, atualizações, operações e mais.

## Quick start

```bash
git clone https://github.com/Dragoon0x/founder-ops.git ~/.claude/skills/founder-ops
cd ~/.claude/skills/founder-ops
cp config/founder.example.yml config/founder.yml
# Edite os 4 arquivos de config com seu contexto
claude
/founder-ops
```
