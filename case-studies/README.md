# Case Studies

Anonymized scenarios drawn from operator experience. Each case is
structured the same way so you can scan for patterns.

## Format

Every case study has:
- **Setup**: stage, team size, revenue, runway, the one-line situation
- **What happened**: the decision and the actions taken
- **Outcome**: what actually happened next
- **What I'd do differently**: the post-mortem
- **Generalizable lesson**: the pattern, not the anecdote
