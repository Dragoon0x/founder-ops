# Default alive, default dead

> Paul Graham's question: at your current growth rate, do you reach profitability before you run out of money? If yes, default alive. If no, default dead.

## The principle

- It doesn't matter what you feel about the business. Run the math.
- If you're default dead, you have two choices: grow faster or cut burn. Probably both.
- If you're default alive, you have the luxury of optionality: raise on your terms, grow at your pace, hire for depth not velocity.
- The state can change in a quarter. Re-run the math monthly.

## Signals it's time to act

- Being default alive doesn't mean you must be profitable. It means you have a path there at current trajectory.
- Investors respect default-alive founders more because they're negotiating from strength, not need.

## Notes

- Default-dead founders often don't know they're default-dead. The math is quiet until it isn't.
