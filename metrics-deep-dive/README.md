## Rules for metrics

1. Every metric has a denominator. Know it.
2. Metrics reported to the board must be reproducible from source.
3. Change the metric definition = flag it and show both old and new.
4. Vanity metrics (total signups, total users) are fine for press, never for decisions.
5. No metric is good in isolation. Always show 3+ together.
