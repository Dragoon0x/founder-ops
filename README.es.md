# Founder-Ops

> **[🇪🇸 Español]** · [English](README.md) · [Português](README.pt-BR.md) · [日本語](README.ja.md) · [한국어](README.ko-KR.md) · [中文](README.zh-CN.md)

**Sistema operativo impulsado por IA para fundadores.**

40 modos especializados que cubren todo el trabajo del fundador: descubrimiento
de clientes, priorización, fundraising, crecimiento, contratación, finanzas,
posicionamiento, contenido, actualizaciones, operaciones, y más.

## Instalación rápida

```bash
git clone https://github.com/Dragoon0x/founder-ops.git ~/.claude/skills/founder-ops
cd ~/.claude/skills/founder-ops
cp config/founder.example.yml config/founder.yml
# Edita los 4 archivos de config con tu contexto
claude
/founder-ops
```

## Filosofía

1. **El tiempo del fundador es el recurso más escaso.**
2. **Decisiones > Entregables.**
3. **Mata el feature creep.**
4. **Evidencia sobre opinión.**
5. **Adversarial por defecto.**

Cada recomendación incluye: confianza 1-10, el argumento más fuerte en contra,
y qué cambiaría tu opinión.
