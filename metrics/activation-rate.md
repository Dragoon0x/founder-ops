# Activation Rate

## Definition
The percentage of new users/accounts who complete a defined set of actions that correlate with long-term retention.

## Formula
```
Activation Rate = (Users who completed activation criteria / Total new signups) × 100
```

Within a defined time window (typically 7 or 14 days from signup).

## Defining your activation criteria
Activation is NOT just "created an account." It's the moment a user gets real value.

### How to find your activation moment
1. Pull a cohort of users who retained at 90 days
2. Pull a cohort of users who churned within 30 days
3. Compare their behavior in the first 7 days
4. Identify the actions that most differentiate retained vs churned users

### Examples by product type
| Product Type | Activation Criteria | Time Window |
|-------------|-------------------|-------------|
| Project management | Created a project + invited 1 team member + completed 1 task | 7 days |
| Analytics tool | Connected data source + created first dashboard | 14 days |
| Communication tool | Sent 5 messages + joined 2 channels | 3 days |
| E-commerce SaaS | Added first product + processed first order | 14 days |
| CRM | Imported contacts + logged first activity | 7 days |
| Design tool | Created first design + exported or shared it | 7 days |

## Benchmarks
| Product Type | Median Activation Rate | Top Quartile |
|-------------|----------------------|-------------|
| PLG B2B | 20-30% | > 40% |
| Sales-led B2B | 60-70% | > 80% |
| Consumer | 10-20% | > 30% |
| Mobile app | 15-25% | > 35% |

Sales-led is higher because customers are pre-qualified and hand-held through setup.

## Improvement levers
1. **Reduce time to value**: Fewer steps between signup and first value moment
2. **Progressive onboarding**: Don't dump all features on day 1
3. **Templates/presets**: Let users start from something, not a blank canvas
4. **Onboarding emails**: Drip sequence guiding users to activation criteria
5. **In-app guidance**: Tooltips, checklists, contextual prompts
6. **Remove friction**: Every form field, confirmation step, or page load is a dropout point

## Alert thresholds
- Activation rate declining MoM for 3+ months
- Activation rate < 20% for PLG product (your onboarding is broken)
- Time to activation increasing (users taking longer to get value)
- Drop-off analysis showing > 50% dropout at any single step
