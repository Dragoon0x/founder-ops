# SaaS Magic Number

## Definition
A measure of sales efficiency: how much ARR you generate for every dollar spent on sales and marketing.

## Formula
```
Magic Number = (Current Quarter ARR - Previous Quarter ARR) / Previous Quarter S&M Spend
```

Alternative (annualized):
```
Magic Number = (QoQ Net New ARR × 4) / Previous Quarter S&M Spend
```

## Interpretation
| Magic Number | Meaning | Action |
|-------------|---------|--------|
| > 1.0 | Efficient. Every $1 in S&M generates > $1 in ARR | Invest more aggressively in S&M |
| 0.75 - 1.0 | Good efficiency | Continue current spending, optimize |
| 0.5 - 0.75 | Moderate | Investigate channel efficiency, may need to optimize before scaling |
| < 0.5 | Inefficient | Reduce spend, fix conversion or product before scaling |

## How it differs from CAC
- CAC tells you the cost to acquire one customer
- Magic Number tells you the overall efficiency of your go-to-market machine
- Magic Number is better for board-level conversations because it's simpler and more holistic

## Limitations
1. Lagging indicator: S&M spend today produces results 1-2 quarters later
2. Doesn't account for churn (a high magic number with high churn is misleading)
3. Quarterly fluctuations can be noisy (use trailing 2-quarter average)
4. Doesn't differentiate between new and expansion ARR

## Alert thresholds
- Magic number < 0.5 for 2+ consecutive quarters
- Magic number declining while S&M spend is increasing (throwing money at a broken funnel)
