# Founder-Ops

> [English](README.md) · [Español](README.es.md) · [Português](README.pt-BR.md) · [日本語](README.ja.md) · **[🇰🇷 한국어]**

**창업자를 위한 AI 기반 운영 시스템.**

고객 발견, 우선순위 지정, 자금 조달, 성장, 채용, 재무, 포지셔닝,
콘텐츠, 업데이트, 운영 등 창업자의 모든 업무를 커버하는 40개의
전문 모드.

## 빠른 시작

```bash
git clone https://github.com/Dragoon0x/founder-ops.git ~/.claude/skills/founder-ops
cd ~/.claude/skills/founder-ops
cp config/founder.example.yml config/founder.yml
# 4개의 설정 파일을 당신의 컨텍스트로 편집
claude
/founder-ops
```
