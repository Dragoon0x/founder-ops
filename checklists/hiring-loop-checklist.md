# Hiring Loop Checklist

Use for every candidate who hits a full loop (Screening passed, moving to
interview panel + work sample + references).

## Pre-Loop (Recruiter or Founder)

### Role prep
- [ ] Role definition written (90-day and 12-month outcomes)
- [ ] Scorecard defined (5-7 dimensions, what great looks like for each)
- [ ] Compensation range decided + approved by you
- [ ] Interview panel assembled (3-5 interviewers for IC; 5-7 for senior)
- [ ] Each interviewer has ONE focus area (don't duplicate)

### Interviewer prep
- [ ] Each interviewer gets:
  - The role definition
  - The scorecard they'll use
  - The candidate's resume + any prior notes
  - The question bank for their focus area
  - Time block in calendar (no double-book)

## Screening Call (30-45 min, founder or hiring manager)

### Questions to cover
- [ ] Why are you open to new roles right now?
- [ ] What's the hardest thing you've built / shipped?
- [ ] What do you know about us? (Did they research?)
- [ ] 2-3 calibration questions specific to role
- [ ] Compensation expectations (approximate, not exact)
- [ ] Timeline + competing processes

### Post-call (within 24h)
- [ ] Scorecard filled out
- [ ] Decision: advance / pass / waitlist
- [ ] If pass: personal email with specific feedback (not boilerplate)
- [ ] If advance: schedule next step within 3 business days

## Work Sample (4-8 hours, paid $500-1500)

- [ ] Problem definition sent (real problem from your company, context-rich)
- [ ] Time expectation stated explicitly
- [ ] Rubric for evaluation shared (or kept internal — decide based on role)
- [ ] Compensation agreement signed before starting
- [ ] Deadline set (48-72 hours to submit)

### Evaluation
- [ ] Quality of output (meets spec)
- [ ] Process visible (how did they think through ambiguity?)
- [ ] Edge cases handled
- [ ] Documentation / communication of work
- [ ] Would you be happy if this was merged into production?

## Interview Panel (3-5 sessions)

### Common failures to avoid
- [ ] No interviewer asks the same question
- [ ] Each interviewer gets SPECIFIC focus area
- [ ] 15 min between interviews for notes
- [ ] Interviewers don't discuss candidate during loop

### Per interviewer
- [ ] Scorecard submitted within 24h of interview
- [ ] Specific evidence (quotes, behavior, output) for each rating
- [ ] Clear hire/no-hire vote with reasoning
- [ ] 2+ specific strengths
- [ ] 2+ specific concerns
- [ ] Questions to follow up in later interviews (if advancing)

## Debrief Meeting

- [ ] All interviewers present (or written vote if they can't attend)
- [ ] Each interviewer shares vote + evidence BEFORE hearing others
- [ ] Hiring manager + panel discuss divergences
- [ ] Final decision: HIRE / NO HIRE / NEEDS MORE DATA
- [ ] If NEEDS MORE DATA: specific next step (additional interview, reference dig, work sample extension)

### The "Would you hire them for anything?" Test
- [ ] Ask each interviewer: "If this exact role didn't exist, would you still hire this person to work at our company?"
- [ ] If most answer NO, it's a NO regardless of scorecard averages

## References (before offer)

- [ ] Minimum 3 forward references + 2 backchannel
- [ ] Each reference gets structured call (see `prompts/reference-check-script.md`)
- [ ] Synthesis note written: consistencies + divergences
- [ ] Red flags explicitly flagged

## Offer

- [ ] Legal / HR review of offer terms
- [ ] Equity grant approved (if applicable)
- [ ] Verbal offer FIRST (phone call — founder or hiring manager)
- [ ] Written offer within 24h
- [ ] Decision deadline: 5-7 business days
- [ ] Check-in calls offered during decision period (not desperate, but open)

## Close

- [ ] Offer accepted: start date confirmed
- [ ] Background check initiated (if applicable)
- [ ] Pre-onboarding kit sent
- [ ] Equipment ordered (shipped to arrive 3 days before start)
- [ ] Access requests submitted (tools, Slack, etc.)
- [ ] Start date team announcement prepped
- [ ] First-week plan drafted (see `checklists/new-hire-first-week.md`)

## If Offer Declined

- [ ] Debrief call (what swayed them?)
- [ ] Debrief notes → document lessons
- [ ] Re-evaluate comp, pitch, or role if pattern emerges (2+ decline for similar reason)
- [ ] Respectful goodbye; invite future conversation

## Red Flags During Loop
- [ ] Candidate no-shows without communication → hard pass
- [ ] Repeated rescheduling → yellow flag, diagnose
- [ ] Hostility in interview → hard pass
- [ ] Hostility with recruiter/coordinator → hard pass (biggest tell)
- [ ] References go silent / hedge → red flag, dig deeper

## Green Flags During Loop
- [ ] Asks sharp questions about business, not just role
- [ ] Has done homework on you / product
- [ ] Pushes back thoughtfully (not aggressively) on something you said
- [ ] Demonstrates they've thought about the role, not just about "a job"
- [ ] References volunteer specific, concrete praise

## Post-Hire

- [ ] 30-day check-in: is scorecard holding up?
- [ ] 60-day check-in: any blind spots from interview emerging?
- [ ] 90-day check-in: would you hire this person again?
- [ ] Hiring lessons captured for next loop
