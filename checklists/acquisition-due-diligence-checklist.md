# Acquisition Due Diligence Checklist (Being Acquired)

Your company is being acquired. The acquirer will do deep diligence.
Being prepared compresses the timeline from 8 weeks to 4.

## Pre-LOI (Letter of Intent)

### Before anything formal
- [ ] NDA signed (mutual)
- [ ] Teaser deck shared
- [ ] Management meeting (acquirer leadership meets yours)
- [ ] Rough valuation discussed

### Engage advisors
- [ ] M&A lawyer (not your regular counsel)
- [ ] Banker (if deal size warrants, typically >$50M)
- [ ] CFO or fractional CFO for diligence support
- [ ] Key employees aware at high level (not detailed until LOI)

## LOI Phase

### Reviewing LOI
- [ ] Price / structure (cash vs. stock)
- [ ] Exclusivity period (30-60 days typical)
- [ ] Closing conditions
- [ ] Key employee retention terms
- [ ] Earn-out if any

### Decision
- [ ] Sign LOI (non-binding except exclusivity typically)
- [ ] Lawyer reviewed
- [ ] Board informed

## Diligence Room Setup (First Week)

### Legal / Corporate
- [ ] Certificate of Incorporation + amendments
- [ ] Bylaws + amendments
- [ ] Stockholder agreements
- [ ] Board minutes (all of them — go back to founding)
- [ ] Cap table (FULLY CLEAN — every share accounted for)
- [ ] Option plan + grants
- [ ] Stock purchase agreements (founders)
- [ ] Prior financing documents (SAFEs, notes, priced rounds)

### Financial
- [ ] Historical financials (3-5 years monthly P&L, BS, CF)
- [ ] Revenue by customer (detailed)
- [ ] Revenue recognition policy (if complex)
- [ ] Monthly bank reconciliations
- [ ] AR aging
- [ ] Deferred revenue schedule
- [ ] Sales tax compliance documentation
- [ ] Payroll records
- [ ] Tax returns (federal + state)
- [ ] Budget vs. actual variance
- [ ] Forward financial model

### Customers
- [ ] All customer contracts (searchable)
- [ ] Revenue concentration analysis (top 5, top 10 as % of ARR)
- [ ] Churn analysis (who left, when, why)
- [ ] Pipeline snapshot
- [ ] Customer quality signals (NPS, health scores)
- [ ] Change-of-control provisions in contracts (critical!)

### Product / Technology
- [ ] Architecture docs
- [ ] Product roadmap
- [ ] Technical debt assessment (honest)
- [ ] Code ownership / IP assignments (every contributor)
- [ ] Open source licenses used (compliance check)
- [ ] Patents + trademarks
- [ ] Third-party licenses (any restrictions?)
- [ ] Security assessments (SOC 2 reports, pen tests)

### People
- [ ] Employee roster with role, comp, equity
- [ ] Employment agreements (all of them)
- [ ] Contractor agreements + IP assignment
- [ ] Employee handbook / policies
- [ ] Org chart
- [ ] Open roles
- [ ] Benefits plans

### Commercial
- [ ] Vendor contracts (all material ones)
- [ ] Insurance policies
- [ ] Litigation (past, present — even threats)
- [ ] Regulatory licenses / compliance

## Diligence Phase (Weeks 2-6)

### Data room management
- [ ] Virtual data room (DealRoom, Firmex, iDeals)
- [ ] Tiered access (some docs Phase 1, deeper Phase 2)
- [ ] Q&A tracker (log every question + answer)
- [ ] Response SLA (24h for standard, 4h for urgent)

### Management meetings
- [ ] CEO + CFO + CTO available
- [ ] Product demo for acquirer leadership
- [ ] Tech deep dive (your CTO + theirs)
- [ ] Customer reference calls (arranged by you)
- [ ] Team interviews (they'll want to meet key people)

### Specific diligence areas that sink deals
- [ ] Customer concentration (>25% in one customer = red flag)
- [ ] Founder IP assignment (all contributors — even that friend who helped 3 years ago)
- [ ] Change of control in customer contracts (prevents auto-termination post-deal)
- [ ] Non-compete with previous employers (founders)
- [ ] Undisclosed liabilities (lawsuits, unpaid taxes, etc.)

## Definitive Agreement Phase (Weeks 6-10)

### Key terms negotiated
- [ ] Purchase price (cash + stock split, if any)
- [ ] Escrow (typically 10-15% held back for 12-18 months)
- [ ] Indemnification (for breaches of reps and warranties)
- [ ] Working capital target
- [ ] Employee retention (who stays, for how long, with what incentive)
- [ ] Non-compete for founders (typically 2-3 years)
- [ ] Closing conditions

### Representations & Warranties (major section)
- [ ] Review each rep carefully (you're swearing these are true)
- [ ] Disclosure schedules (where truth differs from rep, disclose)
- [ ] Knowledge qualifier ("to seller's knowledge" vs. absolute)

## Closing

### Employee communication
- [ ] Day of signing: written announcement to team
- [ ] Individual 1:1s with key employees
- [ ] Retention packages discussed / signed
- [ ] Concerns heard + addressed

### External comms
- [ ] Customer notification (personal for top accounts)
- [ ] Investor notification
- [ ] Press release coordination
- [ ] LinkedIn / social strategy

### Mechanical closing
- [ ] Cap table updates
- [ ] Wire transfer instructions
- [ ] Stock certificates or electronic transfers
- [ ] Closing documents executed

## Post-Close (First 90 Days)

### Integration
- [ ] Integration plan (likely already drafted during diligence)
- [ ] Cultural integration rhythm
- [ ] Technical integration milestones
- [ ] Customer transition plan

### Retention
- [ ] Key employees: weekly check-ins
- [ ] Retention bonuses tracked
- [ ] Culture changes communicated

### Your role
- [ ] Clear scope of responsibility
- [ ] Reporting relationship
- [ ] Departure plan if earn-out / transition period ends

## Red Flags (For You, During the Process)

- [ ] Acquirer keeps moving goalposts on price
- [ ] Diligence requests get invasive / personal (not about business)
- [ ] Team integration lead is evasive or non-committal
- [ ] Non-compete drafts get more restrictive over time
- [ ] Escrow / holdback increases over time
