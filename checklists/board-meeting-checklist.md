# Board Meeting Checklist

For every regular board meeting. More rigorous than you think you need.

## T-14 Days (Planning)

- [ ] Date + time confirmed with all directors
- [ ] Agenda draft in your doc
- [ ] Identify: this meeting's key decision(s) / outcome(s)
- [ ] Identify: sensitive topics to raise (don't bury them)
- [ ] Chair / lead director consulted on agenda

## T-7 Days (Pre-Read Prep)

- [ ] Metrics for the period pulled + verified
- [ ] CEO update drafted (2 pages, prose + key numbers)
- [ ] Deep-dive topic chosen + 1-page memo drafted
- [ ] Questions for the board (specific, written)
- [ ] Financials current (P&L, cash, runway)

## T-3 Days (Materials)

- [ ] Board deck drafted (12-15 slides, see `prompts/board-meeting-prep.md`)
- [ ] Pre-read memo (2-3 pages, prose)
- [ ] Any ad-hoc attachments (customer metrics, org chart, roadmap)
- [ ] All in one shared folder; email link to board

## T-2 Days (Distribute)

- [ ] Pre-read distributed (must be at least 48h before meeting)
- [ ] Individual follow-ups to directors who might have concerns
- [ ] Legal / company secretary reviewed any resolutions / votes

## T-1 Day (Prep)

- [ ] Dry run with co-founder / CFO
- [ ] Anticipated questions + answers
- [ ] Agenda time-boxed (each section strict time)
- [ ] Meeting logistics: room, video, recording, note-taker

## Meeting Day

### Pre-Meeting (15 min)
- [ ] Tech working (video, audio, screen share)
- [ ] Refreshments / comfort (in-person)
- [ ] Notes doc ready
- [ ] Dial-in numbers ready for remote attendees

### During Meeting
- [ ] Stick to agenda + time boxes
- [ ] Capture decisions explicitly ("So we're deciding to do X — everyone agree?")
- [ ] Capture action items with owners + dates
- [ ] If tension arises: name it, don't dodge
- [ ] Votes taken formally for formal items (equity grants, option pool, etc.)

### Closing
- [ ] Next meeting date confirmed
- [ ] Action items reviewed
- [ ] Specific asks to directors between meetings
- [ ] Any executive session needed? (Directors-only, no CEO)

## T+1 Day (After)

- [ ] Draft minutes sent to directors
- [ ] Action items entered in your tracker with due dates
- [ ] Asks documented (what did each director promise to do?)
- [ ] Resolutions prepared for signing (if any)
- [ ] Team update (what the board discussed, at appropriate abstraction)

## T+7 Days

- [ ] Minutes finalized (corrections applied)
- [ ] Minutes signed + stored in company records
- [ ] Action items 1-week check-in
- [ ] Individual follow-up with each director on their promises

## Sensitive Situations Needing Extra Care

### Board considering replacing CEO
- Legal counsel informed
- Formal process documented
- Executive session protocols

### Major financing decisions
- Lead director coordinated
- Conflicts of interest declared
- Votes recorded explicitly

### Performance issues with directors
- Chair conversation first
- Board composition review triggered

## The Meeting Effectiveness Test

After each meeting, rate (internally, privately):
- Did we make the decisions we needed? (yes/no/partial)
- Did the right people speak up?
- Did we waste time on things that should've been async?
- Do directors know what they need to do before next meeting?

If most answers are bad, redesign the format.
