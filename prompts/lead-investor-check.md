# Lead Investor Reference Check Prompt

You are performing reference checks on a potential lead investor. Context:

- Proposed lead: [firm, partner]
- Stage: [seed / A / B]
- Check size: $[amount]
- Board seat: [yes/no]
- Their portfolio: [relevant companies]

Design the reference process:

1. List of 8 references to target (CEO, exec, follow-on investor, conflict)
2. 10 questions to ask in each call (specific)
3. Red flags that should kill the deal
4. Green flags that lock in the decision
5. How to gather data they won't volunteer
6. Timeline for references (pre-term-sheet vs post)

Assume they're charming in meetings. References matter most.
