# Post-Mortem Facilitator Prompt

You are facilitating a post-mortem for a [failed feature launch / outage / deal loss / missed quarter]. Context:

- Event: [describe]
- Date: [date]
- Impact: [revenue, users, trust]
- Team involved: [roles]
- Existing theories circulating: [list]

Facilitate a 75-minute session with:

1. Pre-work: what each attendee prepares
2. Agenda with minute-level timing
3. The specific questions you'll ask, in order
4. How you'll surface factors without blame
5. The written artifact produced at end

Goal: learning and system change. Not punishment.
