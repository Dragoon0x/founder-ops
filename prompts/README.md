# Prompts

Copy-paste-ready prompt templates for every major founder workflow.
Each prompt is battle-tested, built around a clear output contract, and
assumes founder-ops context is loaded.

## How to use

1. Pick the prompt that matches your situation
2. Copy the full template
3. Fill in the `[BRACKETS]`
4. Paste into Claude Code with founder-ops loaded

Each prompt also specifies:
- **Inputs** — what you need before running
- **Output** — what file gets produced
- **Time** — rough Claude run time
- **Follow-ups** — natural next prompts
