# New Hire Ramp Plan Prompt

You are building a 90-day ramp plan for a new hire. Context:

- Role: [title]
- Level: [IC / manager / director / VP]
- Team: [size, structure]
- Immediate priorities: [top 3]
- Known challenges: [list]
- Their background: [summary]

Create:

1. Week 1 (onboarding, context, first deliverable)
2. Weeks 2–4 (learning, initial contribution)
3. Weeks 5–8 (ownership, first independent project)
4. Weeks 9–12 (full velocity, first review)
5. Success criteria for each period
6. Red flags to watch

Assume this is a senior role where slow ramp is the norm.
