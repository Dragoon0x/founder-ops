# Prompt: Landing Page Draft

## Output
`output/landing-page-[product].md`

---

## Prompt Template

```
I need a landing page draft for [PRODUCT].

Context:
- What the product does: [ONE SENTENCE]
- Core value prop: [TRANSFORMATION THE USER GETS]
- ICP: [SPECIFIC — role + company profile + situation]
- Main competitor(s): [NAMES]
- Key differentiator (the thing only you do): [SPECIFIC CAPABILITY OR APPROACH]
- Top 3 customer-reported outcomes: [CONCRETE RESULTS]
- Pricing: [HOW YOU CHARGE]

Run /founder-ops content landing-page and produce:

## HERO
- Headline (10 words max, outcome-focused, NOT feature-focused)
- Subhead (20-30 words, specific about who + what + proof)  
- Primary CTA (action + value)
- Secondary CTA (low-friction alternative)

Write 3 variants of the headline + subhead.

## SOCIAL PROOF STRIP
- Logos to include (top customer logos)
- One-sentence from a real user quote (if we have one; leave placeholder otherwise)

## PROBLEM / SOLUTION
2-3 paragraphs. No features. Describe the user's day with and without.

## HOW IT WORKS (3 steps)
- Step 1: [action + outcome]
- Step 2: [action + outcome]
- Step 3: [action + outcome]

Each step: 1 line of copy + placeholder for a screenshot.

## FEATURES (max 6)
For each:
- What it does (not how)
- Who it's for
- 1-line benefit

## PROOF
- 3 customer quotes (placeholders with role + company)
- 2-3 metrics/stats if we have them

## PRICING PREVIEW
- Short snippet pointing to pricing page
- NOT the full pricing grid on landing page

## FAQ (6-8)
Address the real objections:
- Is this a replacement for [competitor]?
- How long to onboard?
- What about security / compliance?
- What if it doesn't work for us?
- How's this different from [obvious alternative]?

## FINAL CTA
Different framing from hero CTA. More specific.

VOICE:
- Direct, specific, unpretentious
- No adjectives without evidence ("robust", "seamless", "intuitive" → cut)
- No fake urgency
- Second-person ("you") not first-person plural ("we")

Save as output/landing-page-[PRODUCT].md.
```
