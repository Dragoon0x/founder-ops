# Team Offsite Workflow

## Trigger
Bi-annual (or after major milestones: fundraise, reorg, pivot).

## Planning Timeline

### T-6 weeks: Define the purpose
An offsite without a clear purpose is an expensive vacation. Pick ONE primary outcome:
- Strategic alignment (setting direction for next 6-12 months)
- Team bonding (rebuilding trust after hard period)
- Problem-solving (tackling a specific challenge together)
- Planning (quarterly/annual planning in depth)

### T-4 weeks: Logistics
- **Location**: Within 2 hrs travel for most of the team. Avoid resorts that signal excess.
- **Duration**: 2 days is the sweet spot. Day 1 = work, Day 2 = work AM + social PM.
- **Budget**: $500-1,500 per person (travel + accommodation + meals + activities)
- **Agenda**: Block out sessions. No more than 6 hrs of structured content per day. Breaks matter.

### T-2 weeks: Pre-work
Send attendees:
- The strategic context memo (where we are, what's changed, what's ahead)
- 2-3 pre-read documents relevant to the offsite theme
- One question to think about: "What's the most important thing for us to get right in the next 6 months?"
- Logistics: travel details, dress code (casual), dietary requirements

### T-1 week: Final prep
- Facilitator identified (external if budget allows, internal if < 15 people)
- Materials printed (yes, paper. Screens off during sessions)
- Dinner reservation made
- Activity booked (something physical and mildly competitive)

## Offsite Agenda Template (2 days)

### Day 1: Alignment
| Time | Session | Format |
|------|---------|--------|
| 9:00 | Welcome + check-in | Round-robin: "What are you most excited/nervous about?" |
| 9:30 | State of the company | CEO presents (30 min), Q&A (30 min) |
| 10:30 | Break | |
| 10:45 | Customer voice | Play 3-4 customer interview clips, discuss patterns |
| 12:00 | Lunch | |
| 13:00 | Strategic choices | Small groups tackle specific strategic questions |
| 15:00 | Break | |
| 15:15 | Readouts + debate | Groups present, challenge each other's thinking |
| 17:00 | End of structured day | |
| 18:30 | Team dinner | No work talk at dinner. Seriously. |

### Day 2: Action
| Time | Session | Format |
|------|---------|--------|
| 9:00 | Synthesis | Facilitator presents themes from Day 1 |
| 9:30 | Priorities | Vote on top 3-5 priorities for next 6 months |
| 10:30 | Break | |
| 10:45 | Action planning | Each priority gets an owner, milestones, and dependencies |
| 12:00 | Lunch | |
| 13:00 | Team activity | Something physical. Hiking, cooking class, escape room. |
| 16:00 | Closing | Each person: "One thing I'm committing to" |
| 16:30 | Depart | |

## Post-Offsite (within 1 week)
- Share notes with entire company (transparency)
- Publish the priorities list with owners
- Schedule first check-in (2 weeks out)
- Send thank you + feedback survey (3 questions max)

## Common Offsite Failures
1. **All talk, no action**: Great discussion, nothing changes. Fix: every session ends with assigned action items.
2. **CEO monologue**: Offsite becomes a long presentation. Fix: CEO talks < 30% of total time.
3. **Conflict avoidance**: People are "nice" instead of honest. Fix: structured debate formats with clear prompts.
4. **Too packed**: No breathing room. Fix: 50% structured, 50% unstructured/social.
