# Playbook: Start Fundraising

## When to Run
3-6 months before you need cash. Earlier is better.

## The Pre-Work (4-6 weeks before outreach)

### Step 1: Get the Story Right
Run `/founder-ops positioning`
- Refine the elevator pitch
- Test the narrative on 5 friendly people
- Update the deck with the new story

Run `/founder-ops compete`
- Build the "why now" case
- Map the market clearly
- Have your competitive answer locked

### Step 2: Get the Numbers Right
Run `/founder-ops metrics`
- 18-month historical (monthly cadence)
- Cohort retention curves
- CAC by channel with payback periods

Run `/founder-ops finance`
- Build 18-month forecast
- Three scenarios (conservative, base, aggressive)
- Calculate use of funds for the round

### Step 3: Build the Materials
Run `/founder-ops content` (deck mode)
- Pitch deck (10-15 slides max)
- Data room contents
- One-pager / executive summary
- Demo script (5 min crisp)

### Step 4: Build the Pipeline
Run `/founder-ops fundraise` (research submode)
- Identify 30-50 target investors (research each thoroughly)
- Categorize: high priority, medium, fillers
- Map warm intro paths

## The Outreach (2-4 weeks)

### Step 5: Soft Signal
Tell 5-10 trusted advisors / current investors:
"We're going to start raising in [month]. Who should we definitely meet?"

### Step 6: Warm Intros First
Run `/founder-ops fundraise` (outreach submode)
For each warm intro:
- Personalized request to introducer (not generic)
- Forwardable email (clean, professional)
- Calendar link ready

### Step 7: Cold Outreach (in parallel)
For investors without warm path:
- Personalized cold email (not template)
- Wait 5-7 days for response
- One follow-up max

## The Process (4-8 weeks of meetings)

### Step 8: Meeting Cadence
Run `/founder-ops calendar` (meeting prep submode) before EVERY meeting
- Read their last 3 portfolio announcements
- Read their most recent writing
- Have 3 questions to ask THEM

### Step 9: After Each Meeting
Within 24 hours:
- Send thank you + deck (if not already)
- Update `data/investors.md` with notes
- If interested: confirm next step + timeline
- If pass: ask why (most valuable info)

### Step 10: Track the Process
Run `/founder-ops fundraise` (pipeline submode) weekly:
- How many in active conversation?
- Where is each (intro, first meeting, partner meeting, term sheet)?
- What's the conversion rate at each stage?
- What's our timeline pressure?

## The Close (1-3 weeks)

### Step 11: Term Sheet Negotiation
- Get a lawyer (specific to fundraising)
- Compare terms across offers
- Prioritize: valuation, board composition, control terms, pro rata

### Step 12: Signed Term Sheet → Diligence
- Data room access
- Customer reference calls
- Product/tech deep dives
- Background checks

### Step 13: Closing
- Final docs review
- Wire instructions
- Update cap table
- Run `/founder-ops update` with the news (internally first, then externally)

## Output Artifacts
- `output/pitch-deck-v[N].md`
- `output/one-pager.md`
- `data/investors-pipeline.md`
- `output/fundraise-tracker.md`
- `reports/investor-meeting-notes-[fund].md`

## Common Failure Modes
- Not enough investors in pipeline (need 30-50, run 5-10 in parallel)
- Don't track the process (lose deals to follow-up gaps)
- Get attached to one term sheet (lose negotiating leverage)
- Skip the soft signal phase (no momentum)
- Underprice the round (raise too little, dilute too much)
- Overprice the round (down round risk)
