## Three Scenarios

### Scenario A: Conservative
- Growth: 50% of current rate
- No additional hiring
- Same gross expenses

### Scenario B: Base Case
- Growth: Continue current rate
- Planned hires (per hiring plan)
- Modest expense growth (10%)

### Scenario C: Aggressive
- Growth: 1.5x current rate
- Aggressive hiring (per ambitious plan)
- Higher expenses to support growth

## Triggers (set thresholds)

- If cash drops below $X → cut burn by Y%
- If growth drops below X% for 2 months → reduce hiring
- If runway hits 9 months → start fundraising
- If runway hits 6 months → consider bridge round or aggressive cuts
- If runway hits 3 months → emergency mode
