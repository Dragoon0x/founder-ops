# Board Deck Template

## Slide Structure (12-15 slides)

### Slide 1: Title
Company name, board meeting date, meeting number (e.g., "Board Meeting #7")

### Slide 2: Executive Summary
3 bullet points max:
- Where we are (one metric that captures the state)
- What happened (the headline story of this period)
- What we need (the ask or decision for this meeting)

### Slide 3: Key Metrics Dashboard
| Metric | Last Period | This Period | Target | Status |
|--------|-----------|------------|--------|--------|
| MRR | | | | 🟢/🟡/🔴 |
| MRR Growth (MoM) | | | | |
| Net New Customers | | | | |
| Churn Rate | | | | |
| CAC | | | | |
| Runway (months) | | | | |
| Headcount | | | | |

### Slide 4: Revenue Deep Dive
- MRR bridge: starting MRR → new + expansion - contraction - churn → ending MRR
- Revenue by segment or plan
- Pipeline and forecast for next period

### Slide 5: Product Update
- Key features shipped
- Adoption metrics for recent launches
- Roadmap for next quarter (top 3 priorities only)

### Slide 6-7: What Went Well / What Didn't
- 2-3 wins with quantified impact
- 2-3 misses with root cause and remediation plan

### Slide 8: Team Update
- Hires made, open roles, org chart changes
- Any regrettable attrition and why
- Culture health check

### Slide 9: Financial Summary
- Cash position and runway
- Burn rate trend
- Budget vs actual for major line items

### Slide 10: Market & Competitive
- Notable competitive moves
- Market trends affecting the business
- Customer insights

### Slide 11: Decisions Needed
- 1-2 specific decisions with your recommendation
- Options considered and why you recommend your choice

### Slide 12: Look Ahead
- Top 3 priorities for next period
- Key risks and mitigation plans
- Ask (if any): introductions, advice, resources

## Board Deck Rules
- Total deck: < 20 slides. If you can't tell the story in 20, you're not clear enough.
- No surprises. If a board member learns something negative for the first time in the meeting, you've failed.
- Send 48+ hours before the meeting.
- Spend 80% of meeting time on discussion, not presentation.
