# Code of Conduct

## Our Pledge

We pledge to make participation in our project a welcoming experience for
everyone. We are committed to creating an environment that is respectful,
constructive, and focused on building great tools for founders.

## Standards

### Positive behavior
- Being kind and patient
- Respecting different viewpoints
- Accepting constructive criticism gracefully
- Focusing on what's best for the community
- Showing empathy toward others

### Unacceptable behavior
- Personal attacks
- Harassment of any kind
- Publishing others' private information
- Trolling or inflammatory comments
- Any conduct inappropriate in a professional setting

## Enforcement

Violations can be reported to the maintainer via GitHub issues (public)
or email (private, for sensitive matters).

Consequences for violations:
- First instance: Warning
- Second instance: Temporary ban
- Third instance: Permanent ban

## Scope

This Code of Conduct applies in all project spaces (GitHub, Discord, etc.)
and in public when representing the project.
