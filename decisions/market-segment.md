# Decision: Which Market Segment to Target

## Context
You can't serve everyone. Choosing a segment is choosing a strategy.

## Segmentation dimensions

### By company size
| Segment | Employees | ACV Range | Sales Motion | Support Model |
|---------|-----------|-----------|-------------|---------------|
| SMB | 1-50 | $1-10K | Self-serve / PLG | Email, docs |
| Mid-market | 50-500 | $10-50K | Inside sales | Chat, phone |
| Enterprise | 500-5000 | $50-250K | Field sales | Dedicated CSM |
| Strategic | 5000+ | $250K+ | Named accounts | Dedicated team |

### By industry vertical
Horizontal (any industry) vs Vertical (specific industry).
Vertical first is usually faster to PMF because:
- Narrower problem = easier to solve deeply
- Smaller market = less competition initially
- Industry-specific language builds credibility
- Referral networks are strong within verticals

### By buyer persona
Who writes the check?
- Individual contributor (bottom-up, PLG)
- Team lead (mid-level, team plan)
- VP/Director (top-down, sales-assisted)
- C-suite (executive sale, relationship-driven)

## Selection criteria
Score each segment 1-5 on:

| Factor | Weight | SMB | Mid-market | Enterprise |
|--------|--------|-----|-----------|-----------|
| Willingness to pay | 25% | | | |
| Speed to close | 20% | | | |
| Retention rate | 20% | | | |
| Market size | 15% | | | |
| Competitive intensity | 10% | | | |
| Your team's expertise | 10% | | | |

## Expansion path
The typical pattern:
1. Start narrow (one segment, one vertical, one persona)
2. Dominate that niche (> 30% share or > 100 customers)
3. Expand to adjacent segment
4. Repeat

Moving upmarket (SMB → enterprise) is easier than moving downmarket (enterprise → SMB) because you can add features and services. Removing features and lowering prices is harder.

## Review
- Revisit segment strategy annually
- Trigger: win rate drops below 20% in target segment
- Trigger: > 30% of inbound leads are from a segment you're not targeting (maybe the market is telling you something)
